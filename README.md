# 🧠 Mindshift

A small brain-training game built with SwiftUI, where you play Rock, Paper, Scissors against the rules — sometimes you need to win, sometimes you need to lose.

## How it works

- Each round, the app randomly picks a move: Rock 🪨, Paper ✂️, or Scissors 📃.
- The app also tells you whether you need to **win** or **lose** that round.
- Tap the move you think is correct.
- Answer correctly to score a point, incorrectly to lose one (score never drops below 0).
- The game ends after 10 rounds, showing your final score with the option to play again.

## Built with

- Swift
- SwiftUI
- `@State` for game state management

## Why I built this

This project was built as part of a SwiftUI learning challenge, focused on practicing core concepts: state management, conditional views, enums, and basic game logic.
